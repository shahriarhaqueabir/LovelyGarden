Objective: Translate the "Card Game" metaphor into a functional PC browser interface. We will focus on the Garden Grid (the board) and the Inventory (the hand).

Platform Focus: PC Browser. This allows us to use hover states for plant info tooltips and keyboard shortcuts for rapid management.

1. Component Hierarchy
We will use a modular approach to ensure the UI stays synchronized with the RxDB state.

AppContainer: The root layout.

GardenField: The main 2D grid area.

GridSlot: A droppable square that handles plant placement.

PlantedCard: The visual representation of a growing plant.

InventoryTray: The "Hand" at the bottom of the screen containing available seeds.

PlantInspector: A side-panel or modal that appears when a card is clicked/tapped.

2. The Garden Grid (CSS Grid & DnD)
To achieve the "Card Game" feel on a PC, we utilize dnd-kit for performance and flexibility.

TypeScript
// components/GardenGrid.tsx
import React from 'react';
import { useDroppable } from '@dnd-kit/core';

interface GridSlotProps {
  x: number;
  y: number;
  content?: PlantedCard; // From Agent A schema
}

export const GridSlot: React.FC<GridSlotProps> = ({ x, y, content }) => {
  const { setNodeRef, isOver } = useDroppable({
    id: `slot-${x}-${y}`,
    data: { x, y }
  });

  return (
    <div 
      ref={setNodeRef}
      className={`
        relative w-32 h-44 border-2 rounded-lg transition-colors
        ${isOver ? 'border-green-400 bg-green-900/20' : 'border-stone-700 bg-stone-800/40'}
      `}
    >
      {content ? (
        <PlantedCardView plant={content} />
      ) : (
        <span className="absolute inset-0 flex items-center justify-center text-stone-600 text-xs">
          {x},{y}
        </span>
      )}
    </div>
  );
};

// components/SeedCard.tsx
import { useDraggable } from '@dnd-kit/core';

export const SeedCard: React.FC<{ catalogId: string; name: string }> = ({ catalogId, name }) => {
  const { attributes, listeners, setNodeRef, transform } = useDraggable({
    id: `seed-${catalogId}`,
  });

  const style = transform ? {
    transform: `translate3d(${transform.x}px, ${transform.y}px, 0)`,
    zIndex: 999
  } : undefined;

  return (
    <div
      ref={setNodeRef}
      style={style}
      {...listeners}
      {...attributes}
      className="w-24 h-36 bg-emerald-700 rounded-md shadow-xl border border-emerald-500 cursor-grab active:cursor-grabbing p-2 flex flex-col justify-between"
